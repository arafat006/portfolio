About Me

My name is Md. Arafat Hossain

I'm an expert at full-stack web development, currently working as a full-time Software Engineer at Enosis Solutions. I also develop mobile apps as a hobby. I've additionally worked with Unity Games before from my coding interest.

Programming for me is like playing an exciting game without being bored. I can work for a long term at a stretch. Bug fixing, defining new techniques, designing industry level architecture or strategies are fun to me. I am a technology seeker. So, I have incredible interest and obviously I pay great attention to quality and timing. I am highly motivated, and I can learn quickly.

My main strengths are:

• Clean coding: A very essential term for developers to understand code with less effort. Maintain industry level coding style can be very helpful to grow bigger and bigger application without any hassle.

• Layered architecture: Project architecture plays very important role to understand project components and help future developers.

• Pixel perfect: Each & every component, UI, Layout will be pixel perfect.

• Unit Test: Test all methods to decrease application's bug and make code more readable.

• End To End Test: Testing all end-points to verify security and ensure data passing ability.
